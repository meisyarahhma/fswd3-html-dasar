Hypertext Markup Language (HTML)

Hypertext Markup Language (HTML) merupakan bahasa pemrograman yang digunakan dalam pembuatan halaman website. HTML membantu dalam pembuatan website yang tersusun dari blok yang memakai elemen-elemen HTML, seperti tag dan atribut.
Fungsi HTML yaitu

  1. Membuat struktur halaman web

    HTML berfungsi dalam membangun struktur halaman website, mulai dari header, footer, navigasi, dan kontennya. Sehingga HTML dapat dikatakan sebagai pondasi website dan dasar dalam memahami bahasa pemrograman.

  2. Menambahkan konten website

    Di HTML kita juga bisa menambahkan gambar dan berbagai konten lainnya pada halaman website, seperti video, tabel, dan media lainnya.

  3. Mengatur format dan tata letak

    Dalam HTML kita dapat mengatur format dan tata letak konten dalam website, mulai dari ukuran dan jenis huruf, warna huruf, dan tata letak isi konten website.

  4. Mengarahkan pengguna ke halaman website lainnya

    HTML dapat dirancang agar pengguna bisa mengakses halaman website lain memakai link tertentu, dengan cara disematkan kedalam teks tertentu atau bisa disebut anchor text.


Komponen dalam HTML:
  1. Tag

    Tag merupakan tanda awalan dan akhiran yang terdapat pada perintah HTML yang nantinya akan dibaca oleh browser. Tag dibuat dengan menggunakan kurung siku <â€¦>, di mana di dalamnya berisi nama tag, dan masing-masing tag mempunyai fungsi perintah yang berbeda-beda. Tag dituliskan secaca berpasangan karena terdiri dari tag pembuka dan penutup.

  2. Elemen

    Elemen merupakan komponen HTML yang terdiri dari teks dan simbol yang berupa tag pembuka, isi tag, dan tag penutup. Elemen juga berisi dari elemen-elemen lainnya sehingga isinya tidak hanya satu tag saja, tetapi ada banyak tag.

  3. Atribut

    Atribut merupakan informasi atau perintah tambahan yang terdapat pada elemen, yang berfungsi untuk menjelaskan perintah tag dalam elemen.
